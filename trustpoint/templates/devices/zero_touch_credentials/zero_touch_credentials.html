{% trans "AOKI - Automated Onboarding Key Infrastructure" %}

{% blocktrans %} AOKI (Automated Onboarding Key Infrastructure) is an approach to automatically provision device credentials during the initial device lifecycle phase. It enables zero-touch onboarding by using existing manufacturer-issued IDevID certificates combined with mDNS discovery. {% endblocktrans %}

{% trans "AOKI is available in two protocol variants:" %}

  • {% trans "AOKI with CMP:" %} {% trans "Uses Certificate Management Protocol (CMP) with IDevID certificate-based authentication and IDevID private key signing for request authentication." %}
  • {% trans "AOKI with EST:" %} {% trans "Uses Enrollment over Secure Transport (EST) with mutual TLS (mTLS) authentication using the IDevID certificate for secure channel establishment." %}
