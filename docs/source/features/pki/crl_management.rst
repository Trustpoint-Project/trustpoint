==============
CRL Management
==============

Certificate Revocation List (CRL) management is a critical component of PKI operations. Trustpoint provides tools to manage CRLs effectively.

What is a CRL?
==============

A Certificate Revocation List (CRL) is a list of digital certificates that have been revoked by the issuing Certificate Authority before their scheduled expiration date. CRLs are an essential part of the PKI infrastructure, allowing clients to verify whether a certificate is still valid.
